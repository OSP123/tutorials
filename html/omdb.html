Using the OMDb API

Begin

In this tutorial, we are going to learn how to use the OMDb API: link

  1. First, try reading through the documentation provided in the link above. Notice that we will have to use a url with parameters placed in the url in order to gather the data we need.
  2. Here, we create the HTML for the code.
    3. <form class="form-inline" role="form" id="omdb_form">
  <div class="form-group">
    <label class="sr-only" for="title">Title:</label>
    <input type="text" class="form-control" id="title" placeholder="Title">
  </div>
  <div class="form-group">
    <label class="sr-only" for="year">Year:</label>
    <input type="text" class="form-control" id="year" placeholder="Year">
  </div>
  <button type="submit" class="btn btn-default">Submit</button>
</form>
  3. Next, we're going to add the functionality. We'll use a click function to start off our query.
    4. $("#omdb_submit").click(function(){

});
  4. Next, let's store the title and year values that users input into variables. These variables will then be used in the url string.
  5. Now that we have the variables, we can concatenate them into a query string. This query string is going to be our url that we send to the API.
    6. $("#omdb_submit").click(function(){
    //Gather values from the other fields and store them into variables.
    var title = $("#title").val();
    var year = $("#year").val(); 
    // Concatenate those variables to a query string.
    var queryString = "http://www.omdbapi.com/?t=" + title + "&y=" + year + "&plot=short&r=json";
});
